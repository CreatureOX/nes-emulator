## Introduction
A Nes Emulator implemented by Python

3 Modes support: 
* Pure Python
* Mixed Python & Cython
* Pure Cython  

For more details, check `setup.py`

## Installation
1. `pip install -r requirements.txt`
2. `python setup.py build_ext --inplace`
3. `python .\nes\gui.py`

## Support Games
* Mapper 000 Games 
* Mapper 001 Games
* Mapper 002 Games
* Mapper 003 Games
* Mapper 004 Games
* Mapper 066 Games
